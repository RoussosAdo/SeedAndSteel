# Seed & Steel

Seed & Steel is a top-down 2D action game built in Unity, combining fast combat, progression systems, and a strong visual identity in a handcrafted world.

The project is being developed with a focused production mindset: clean structure, scalable systems, and proper team workflow from the start.

## Overview

The goal of Seed & Steel is to create a polished top-down experience with satisfying movement, responsive combat, and room for deep progression and expansion.

This repository contains the full Unity project, including gameplay systems, scenes, prefabs, scripts, art integration, and production setup.

## Project Status

Currently in active development.

At this stage, the focus is on:

- project architecture
- top-down core movement
- combat foundations
- scene structure
- prefab workflow
- GitHub collaboration pipeline
- scalable Unity setup for a small team

## Tech Stack

- **Engine:** Unity 6
- **Language:** C#
- **Template:** Universal 2D
- **Version Control:** Git + GitHub
- **IDE:** Visual Studio / JetBrains Rider
